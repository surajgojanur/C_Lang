INSTRUCTIONS:
-------------
1. Each problem folder contains:
   - Input file(s) (CSV or Excel)
   - solution.py (main Python script)
   - README.txt (problem description)
   - create_input.py (for Excel-based problems only)

2. For CSV-based problems (1, 2, 5, 6):
   - Input CSV files are already provided
   - Simply run: python solution.py
   - Output files will be generated

3. For Excel-based problems (3, 4, 7, 9, 10):
   - First run: python create_input.py (to create Excel input)
   - Then run: python solution.py
   - Output files will be generated

4. For Problem 8 (Bus Pass):
   - Input CSV is provided
   - Run: python solution.py
   - Generates both CSV and Excel outputs

QUICK START:
------------
Navigate to any problem folder:
  cd EL3/Problem01_Student_Marks
  python solution.py

For Excel problems:
  cd EL3/Problem03_Attendance_Analysis
  python create_input.py
  python solution.py

REQUIREMENTS:
-------------

----------------
1. Student Marks Processing - Calculate average and pass/fail

8. Bus Pass Requests - Calculate fare based on distance
9. Returns Validation - Validate refund mode and amount
10. Electricity Bill Calculator - Slab-based billing
