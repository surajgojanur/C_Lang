Problem 4: Employee Salary Update

Input Files:
- emp_data.xlsx

Output Files:
- emp_salary.xlsx

Logic:
Read employee data from Excel, calculate HRA (10% of basic), DA (18% of basic), compute GrossSalary = Basic + HRA + DA, write only EmpID, Name, GrossSalary to output Excel.

Note: Run create_input.py first to generate emp_data.xlsx
